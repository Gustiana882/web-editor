## 🛠️ Installation Steps

1. Clone the repository

```bash
git clone https://github.com/Gustiana882/web-editor.git
```

2. Install dependencies

```bash
npm install
```

3. Run the app

```bash
npm run dev
```

4. Build app

```bash
npm run build
```
